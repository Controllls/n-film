# CLAUDE.md — 엔필름

> 세션마다 **항상 로드**되는 핵심 규칙. 컨텍스트는 유한 자원이므로 **짧게 유지**한다.
> 여기엔 high-signal 만 남기고 상세는 `docs/` · `.claude/rules/` 에 위임한다.
> 규칙은 **관찰된 실패 패턴**이 생겼을 때만 추가한다 — 미리 채우지 말 것.

## 1. 정체성

- **이름**: 엔필름
- **한 줄**: 촬영 현장 인력 구인구직 하는 사이트
- **사용자**: 감독, 촬영 인력 구인 담당자, 촬영 기숦이 있는 기술자
- **플랫폼**: Web
- **성공 기준**: 공고를 누가 올리고 거기에 사람이 신청하고 , 사람을 다 모으면 성공
- **제품 한 줄**: 감독이 필요한 인원을 필요할떄마다 우리 사이트에 공고글을 올려서 사람을 구한다, 일 이 필요한사람들은 우리사이트에 와서 공고에 신청을한다
- **디자인**: 흰바탕에 살짝 연한 다크(영화 흑백영화정도 색깔)로 오브젝트들을 구성, 적당한 하늘색 첨가

## 3. 핵심 규칙 (타협 불가)

1. **단방향 흐름**: UI → hook → service → backend. 역방향 금지.
2. **Feature-first**: 기능 단위 폴더(`features/<feature>/*`). 외부 노출은 `index.ts` 로만.
3. **I/O 격리**: 화면/훅에서 SDK 직접 호출 금지. service 가 앱 타입으로 변환해 반환.
4. **서버/클라 상태 분리**: 서버 데이터는 서버상태 라이브러리, UI 상태만 글로벌 스토어.
5. **`any` 금지**: 막히면 `unknown` + 가드. 억지로 쓰지 말고 원인 설명 후 질문.
6. **침묵의 실패 금지**: try-catch 로 삼키지 말 것. 의미 있는 메시지로 rethrow.
7. **토큰만 참조**: 색·간격·타이포는 하드코딩 금지, `constants/theme` 토큰만.
8. **사용자에게 기술 용어 노출 금지**: 에러는 한국어 사용자 메시지로.

> 상세(네이밍·파일 크기·주석·타입·에러·상태·테스트) → **`docs/conventions.md`** (필요할 때 읽기)
> UI·디자인·접근성 → **`.claude/rules/components.md`** (UI 파일 작업 시 자동 적용)

## 4. 세션 의식 (Session Ritual)

**시작 — 이걸 끝내기 전 코드 수정 금지:**
1. `git log --oneline -10` + `DEVLOG.md` 의 "다음 할 일" 읽기
2. `PLAN.md` 에서 `[ ]` 미완료 최상단 태스크 **1개** 선택
3. 그 태스크의 `FEATURES.md` F-NN + 관련 `specs/*.md` 읽기 (없으면 먼저 작성·확인)
4. §0 명령으로 빌드/테스트 1회 — 깨진 상태가 아닌지 확인

**종료:**
1. 작동하는 단위로 커밋 (Conventional Commits, "왜" 포함)
2. `PLAN.md` 체크박스 갱신 — **실제 검증한 것만** `[x]`
3. `/log` 로 `DEVLOG.md` 갱신 (변경·결정·기술부채·다음 할 일)

## 5. 무결성 규칙 (Integrity)

- **한 번에 기능 하나.** 끝내고 다음으로. 과욕 금지.
- **테스트·수락기준·스펙을 "통과시키려" 수정·삭제 금지.** 코드가 기준을 만족해야 한다 (역방향 금지).
- **검증 없이 완료 처리 금지.** 실행해서 눈으로 확인. 유닛 통과만으로 `[x]` 금지 → `/verify`.
- **요청된 것만.** 주변 정리·리팩터·독스트링 끼워넣기 금지. ("X 고쳐줘" 엔 X만)
- **스펙과 코드가 다르면** 조용히 맞추지 말고 사용자에게 질문.

---

## 6. Claude 협업 규칙

- **적정 고도**: 지시는 brittle 하드코딩도 막연함도 아닌 중간. 강한 휴리스틱 + 유연성.
- **큰 기능은 먼저 인터뷰**: `AskUserQuestion` 으로 엣지케이스·UX·트레이드오프 질문 후 착수.
- **넓은 탐색은 서브에이전트에 위임**: 여러 파일 훑기는 맡기고 결론(요약)만 받아 컨텍스트 절약.
- **불확실하면 질문.** 추측으로 확장 금지.
- **소통 한국어**, 코드·커밋·식별자는 영어.
