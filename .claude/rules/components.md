# 컴포넌트 · 디자인 · 접근성 규칙

> 경로 한정 규칙(just-in-time). UI 파일을 만질 때만 컨텍스트에 들어온다.
> 스택이 다르면 위 `paths` 를 조정할 것.

## 프레임워크 컴포넌트

- 함수 컴포넌트만. 클래스 금지.
- Props 타입은 `type Props = {...}` 로 명시
- 조건부 렌더는 `&&` 대신 삼항 선호 (`0` 렌더 버그 방지)
- `memo/useMemo/useCallback` 은 프로파일링 후 투입
- 훅은 최상위에서만 (조건·반복 금지)

## 접근성 (a11y)

- 터치 대상 최소 `44×44`
- 이미지/아이콘에 라벨, 슬라이더 등은 현재 값 포함
- 텍스트 대비 WCAG AA
